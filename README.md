# GenQuizz
- This project is a quizz game based on 20 questions on random subjects.

# Prerequisites
- PHP 7

# Usage
- Open a terminal at the root of the repository and type : php -S 127.0.0.1:8080

- Open localhost:8080 in your browser.

- Play !
